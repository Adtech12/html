1) Print First N Natural Numbers

Here's a Python code to print the first N natural numbers:


def print_natural_numbers(n):
    for i in range(1, n + 1):
        print(i, end=" ")

# Example usage:
n = 10
print("First", n, "natural numbers:")
print_natural_numbers(n)

2) Print Table of Desired Choice

Here's a Python code to print the table of a desired choice:


def print_table(number, limit):
    for i in range(1, limit + 1):
        print(number, "x", i, "=", number * i)

# Example usage:
number = 5
limit = 10
print("Table of", number, "up to", limit, ":")
print_table(number, limit)

3) Print Sum of Even Numbers

Here's a Python code to print the sum of even numbers:


def sum_even_numbers(n):
    sum = 0
    for i in range(2, n + 1, 2):
        sum += i
    return sum

# Example usage:
n = 10
print("Sum of even numbers up to", n, ":", sum_even_numbers(n))

4) Print Sum of Digits of Number

Here's a Python code to print the sum of digits of a number:


def sum_of_digits(number):
    sum = 0
    while number > 0:
        digit = number % 10
        sum += digit
        number //= 10
    return sum

# Example usage:
number = 12345
print("Sum of digits of", number, ":", sum_of_digits(number))

5) Check Whether a Number is Odd or Even

Here's a Python code to check whether a number is odd or even:


def check_odd_even(number):
    if number % 2 == 0:
        return "Even"
    else:
        return "Odd"

# Example usage:
number = 10
print(number, "is", check_odd_even(number))
